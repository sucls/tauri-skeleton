### toolchain

> rsutup show : 列出现在使用的和已安装的rust版本。
> rustup update : 更新所有已安装版本，由于nightly偶尔会爆肝日更，所以谨慎更新。
> rustup default: 设置将要使用的版本。
> rustup component <sub> : 列出(list)、安装(add)、移除(remove)组建。
